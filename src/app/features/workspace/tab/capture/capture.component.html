Capture

Start opens a separate browser window. Only that window’s HTTP(S) traffic is listed here (method, URL, status, timing, headers). Response bodies are not recorded—Electron’s supported webRequest API does not expose response bytes. WebSockets are skipped. Authorization and Cookie header values are redacted.

Capture runs in the desktop app (Electron). Use npm run dev or a packaged build to record traffic.
