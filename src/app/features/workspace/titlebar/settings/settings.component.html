Data & config

Collections, settings, session, and response history are stored as JSON under workspace/ in your data directory (plus app-config/ and logs/). Use Change work directory to pick a folder (writes user-data-path.txt under .api-workbench in your profile; the app relaunches automatically to apply it). If API_WORKBENCH_USER_DATA is set, it takes precedence and the change-folder controls are disabled. Bundled repo config/ is not auto-imported for a custom root.
