> **Maintainers:** Sync these pages to the [GitHub Wiki](https://github.com/matthiaskopeinigg/api-workbench/wiki) when publishing user-facing docs. In-app **Help** (activity bar → Help) summarizes a subset; the wiki is the full reference.

## About the app

**API Workbench** is a desktop HTTP/API workbench (Electron + Angular). The UI is a **single workspace window**: there are no separate browser routes for features—everything opens as **tabs** and **sidebar panels** inside one shell.
