# History

> **Maintainers:** Sync to [GitHub Wiki](https://github.com/matthiaskopeinigg/api-workbench/wiki).

## History sidebar

**History** in the activity bar opens a list of **recently sent requests** (method, URL, status, timing). It complements per-request response history inside the request tab.

## Typical actions

- **Open** an entry — opens a tab (or focuses) so you can **resend**, compare, or copy details.
- Entries reflect what was actually sent (resolved URL, etc., depending on implementation).

## Storage

History is stored locally with the workspace; it is not cloud-synced by default.
